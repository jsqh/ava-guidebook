### 2. 文件快捷操作 (File Operations)

工作区顶部的工具栏为开发者提供了高效的资源调度能力：

* **快捷构建**：支持在当前目录下即时创建文件或文件夹，构建项目结构。
* **本地导入**：支持将外部脚本或数据集一键上传至 Agent 的工作目录。
* **状态同步**：手动触发视图刷新，确保 Agent 识别到本地目录的物理变动。
* **即时编辑**：支持代码与 Markdown 的在线改写，通过 `Ctrl+S` 实现本地静默保存与格式校验。
* **环境记忆**：系统自动保存您的折叠/展开习惯，确保下次启动时无缝衔接。
