### 5. 技能管理与自动化 (Skills & Automation)
Ava 的技能系统支持多渠道获取与管理：

* **技能广场与状态控制：** 进入 `技能管理`。您可以在此开启/关闭已安装的技能（如 `memory` 记忆管理、`github_repo_activity` 等）。
* **导入技能：** 点击页面右上角的 **“+ 导入”** 按钮，可直接载入外部封装好的技能文件。
* **对话式技能创建：** 在主对话区，您可以通过自然语言指令（如：“帮我写个脚本并封装成技能”）直接创建新技能。创建成功后，该技能会自动同步出现在 `技能管理` 列表中。
* **💡 操作提示：**
如果您在对话区新创建了一个技能但没有生效，请前往 `设置 -> 技能管理` 确认该技能的开关是否已打开，并记得点击右上角的 **“保存配置”**。


* **创建定时任务：** 进入 `定时任务` -> `新建任务`。
    * 输入 CRON 表达式及希望 Agent 自动执行的提示词。
    * 定时任务亦可在主对话界面通过自然语言交互直接创建。
* **设置快捷指令：** 进入 `快捷指令`。
    * 点击 `添加新命令`。定义关键词（如 `/weekly`）和模板（如 `请帮我整理本周工作大纲: {+}`）。
