## 智能体团队：完整配置流程


1.  **新建仿真**：点击 `智能体团队` -> `新建仿真`。
2.  **基础设置**：
      * **名称与背景**：输入您想建立的团队的名称并设定主题。
      * **策略选择**：设置执行策略（顺序/并行）、最大轮数及参与者数量。
      * **模式**：选择 `多智能体协作`或其他几种模式。
3.  **角色与策略**：
      * 添加并配置专业角色。
      * 自动生成后，您也可以为每个角色单独编辑 Prompt（提示词）及所具备的工具。
4.  **预览与启动**：检查无误后点击 **"启动"**。您可以实时监控各个 Agent 的任务进度与结果，最终成果将自动输出到 Workspace。
